# 광고 클릭 예측 (Advertising Click Prediction)

본 프로젝트는 사용자의 행동과 인구통계 데이터를 기반으로 **광고 클릭 여부를 예측**하는 것을 목표로 합니다.  
통계 분석과 머신러닝 모델링을 통해 **마케팅 전략 수립에 활용 가능한 인사이트**를 제공합니다.

---

## 1. 데이터셋

분석에 사용된 데이터는 1,000명의 사용자 정보를 포함하고 있으며 주요 변수는 다음과 같습니다:

- **사이트 체류 시간**: 사용자가 하루 동안 사이트에서 머문 시간 (분)
- **나이**: 사용자의 연령
- **지역 소득**: 사용자가 속한 지역의 평균 소득
- **하루 인터넷 사용량**: 하루 평균 인터넷 사용 시간
- **성별**: 남성/여성
- **광고 클릭 여부**: 사용자가 광고를 클릭했는지 여부 (타겟 변수)

**기본 통계 요약:**

- 평균 체류 시간: 65분  
- 평균 나이: 36세  
- 평균 지역 소득: 약 5만 5천 달러  
- 클릭 비율: 50% 클릭, 50% 비클릭  

---

## 2. 통계 분석 및 마케팅 인사이트

### 2.1 성별과 광고 클릭

- 분석 결과, 성별과 광고 클릭 사이에는 **유의미한 관계가 없음**  
- **마케팅 인사이트:** 성별 기반 타겟팅은 큰 효과를 기대하기 어렵고, **광고 메시지나 콘텐츠 전략이 더 중요**함  

### 2.2 체류 시간과 광고 클릭

- 광고를 클릭한 사용자의 평균 체류 시간은 클릭하지 않은 사용자보다 **짧음**  
- **마케팅 인사이트:** 체류 시간이 짧더라도 관심을 끌 수 있는 **짧고 임팩트 있는 광고 콘텐츠** 필요  

---

## 3. 데이터 전처리

- **범주형 변수 처리:** 도시와 국가 정보를 활용해 다중공선성을 방지  
- **이상치 제거:** 극단적인 값 제거를 통해 모델 정확도 향상  
- **클래스 균형:** 클릭 여부가 50/50으로 균형 잡혀 있어 데이터 불균형 문제 없음  

---

## 4. 모델링

- **모델:** 로지스틱 회귀(Logistic Regression)  
- **특징 변수:** 체류 시간, 나이, 지역 소득, 인터넷 사용 시간, 성별  
- **모델 성능:**  
  - 정확도(Accuracy): 98%  
  - F1-score: 0.98  

**마케팅 인사이트:** 광고 클릭 예측을 통해 **타겟팅 효율화와 개인화 광고 전략** 가능  

---

## 5. 결론 및 마케팅 전략

1. **성별 기반 타겟팅 효과 미미**  
   - 성별보다는 **사용자 행동 기반 전략**에 집중  

2. **체류 시간이 짧은 사용자도 클릭 가능**  
   - 짧고 강렬한 광고 콘텐츠 개발 필요  

3. **예측 모델 활용 가능**  
   - 광고 클릭 가능성이 높은 사용자를 선별하여 **광고 ROI 최적화** 가능
